# hasura-mqtt

### Run MQTT wrapper
```sh
cpm expample.env .env
# configure MQTT AND HASURA URLS 
yarn 
yarn start
# to start emulator, run the emulator parallelly 
yarn start:emulator 

# or 
npm i
npm run start
```

### Prerequisites

MQTT broker 
Hasura instance
NodeJS and NPM


** package locked with yarn config
