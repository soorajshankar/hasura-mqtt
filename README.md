# hasura-mqtt

### Run MQTT wrapper
```sh
cp expample.env .env
# configure MQTT AND HASURA URLS 
yarn 
yarn start
# to start emulator, run the emulator parallelly 
yarn start:emulator 

# or 
npm i
npm run start
```

### Prerequisites

1. MQTT broker 
2. Hasura instance
3. NodeJS and NPM


** package locked with yarn config
